# Generic Storefront Launch Guide

If you decide not to use Big Cartel and instead use a different platform (like Mixam UK, Shopify, or another generic storefront solution) to sell your physical artefact, follow these steps to link your platform to the game.

## 1. Update the Action Button Link

Regardless of the platform you choose, you need to point users from the ARG to your store.

1. Open `public/secure-data-drop-page.html`.
2. Locate the "PROCURE PHYSICAL ARTEFACT" button. It currently points to a placeholder:
   ```html
   <a href="procure-physical-artefact.html" class="btn-store">PROCURE PHYSICAL ARTEFACT</a>
   ```
3. Replace `"procure-physical-artefact.html"` with the full, live URL of your chosen storefront (e.g., `"https://your-store.com/buy"`).
4. Save the file.

*(Optional)* You may also want to update or delete the `public/procure-physical-artefact.html` placeholder file to ensure users don't accidentally navigate to it directly.

## 2. Aesthetic Considerations for Other Platforms

Unlike Big Cartel, not all platforms allow for easy, global CSS injection. `public/Big Cartel CSS.txt` is specifically written targeting common classes used by Big Cartel templates (like `.site-header`, `.product-list-item`, etc.). If you are using a different platform:

1. **Check for Custom CSS Support:** Determine if your new platform allows custom CSS overrides.
2. **Adapt the Selectors:** If custom CSS is allowed, you will likely need to modify the selectors in `public/Big Cartel CSS.txt` to match the specific HTML structure and class names generated by your new platform.
3. **Core Styles to Port Over:** Even if you can't inject raw CSS, try to customize the platform's native theme settings to match the ARG's visual identity:
   * **Background:** Black (`#050505` or `#000000`).
   * **Primary Text Color:** Neon Green (`#00ff00`).
   * **Accent/Highlight Color:** Crimson Red (`#ff003c`).
   * **Typography:** Use monospace fonts (like `VT323` or generic `Courier New`) for body text, and heavy sans-serif fonts (like `Anton` or `Impact`) for headings and product titles.
   * **Button Styling:** Hard-edged, square buttons (no border radius).

## 3. Verify Storefront Functionality

After linking the site:
* Click the "PROCURE PHYSICAL ARTEFACT" button on the Secure Data Drop page to ensure it correctly opens your external storefront.
* Verify that the transition from the ARG narrative to the store isn't overly jarring, by attempting to match colors and fonts as closely as the platform allows.
