# 3minsto9 ARG / Web Experience

This repository contains the web assets for the "3minsto9" Alternate Reality Game (ARG) and immersive media project.

## About

The files in this repository make up the various interactive websites, puzzles, and hidden "in-universe" pages for the project. The narrative revolves around an underground gang called **Team Rabbit** and their fight against the corporate entity **MLTK** and the **City of Everywhere**.

## Contents

The current assets are stored as raw `.txt` files containing HTML, CSS, and JavaScript. They are meant to be deployed or integrated into a larger web application to form a cohesive ARG experience.

Some of the key files include:

- **Surface Home Page / Velvet Rope Landing Page:** The "front-facing" corporate websites of the City of Everywhere.
- **MLTK Login Gate / NOVA Classified Archive:** Internal corporate pages requiring passwords or puzzles to access.
- **Ollies Radio Scanner / Secure Data Drop Page:** Hidden rebel communication nodes for Team Rabbit.
- **In-Universe 404 Error:** A customized 404 page that hides narrative secrets.
- **Studio Manifesto Page:** Information about the real-world creators, 3minsto9 Operations.
